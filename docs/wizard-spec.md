# ТЗ мастера UTM-генератора (Matrius)

## Общая структура

```
Шаг 0: Компания (Matrius / Zerocoder)
Шаг 1: Тип задачи
  ├── Анонс
  ├── Гайд
  ├── Реклама
  ├── СММ
  └── Свой вариант
```

На каждом шаге — обязательная опция «Свой вариант» для гибкости.

---

## Ветка АНОНС

**Шаг 2А — Платформа** (выбор):
- Email
- Telegram-бот
- VK-бот
- Max-бот

**Шаг 2Б — Конкретный канал** (зависит от платформы):

| Платформа | Канал | utm_source | utm_medium |
|---|---|---|---|
| Email | Unisender | `email` | `unisender` |
| Email | Геткурс | `email` | `getcourse` |
| Telegram-бот | Матриус | `tg-bot` | `schoolmatrius_bot` |
| Telegram-бот | Геткурс | `tg-bot` | `school_matrius_bot` |
| VK-бот | Матриус | `vk-bot` | `matrius_ru` |
| VK-бот | Геткурс | `vk-bot` | `gc_matrius_ru` |
| Max-бот | Матриус | `max-bot` | `id1840120295_bot` |
| Max-бот | Геткурс | `max-bot` | `id1840120295_5_bot` |

**Шаг 3 — Сайт** (выбор из справочника) → `utm_campaign` = **хвост URL** автоматом (path после слеша, или домен если path пустой). Тег из справочника здесь не используется — он показывается только как подсказка для навигации.

**Шаг 4 — Дата** (календарь сегодня → +14 дней) → `utm_term` формат `nn.nn.nn`

**Авто:** `utm_content = announce`

## Ветка ГАЙД

**Шаг 2 — Название документа:**
- Поле ввода названия на русском
- Программа предлагает варианты транслитерации/сокращения
- Пользователь выбирает или вводит свой
- → `utm_medium`

**Шаг 3 — Сайт** (из справочника) → `utm_campaign` автоматом

**Шаг 4 — Место размещения** (кнопки):
- `link-text`
- `qr`
- `image`
- `button`
- Свой вариант
- → `utm_content`

**Шаг 5 — Дата первого запуска** (календарь) → `utm_term` формат `mm.yy`

**Авто:** `utm_source = pdf`

## Хвостик сайта (общая логика)

Парсится из URL:
- `https://matrius.online/` → хвостик = `matrius.online`
- `https://matrius.online/olimpiyadnayamatematika` → хвостик = `olimpiyadnayamatematika`

## Финальный экран

- Поля UTM (source/medium/campaign/content/term)
- Базовый URL (выбранный сайт)
- Собранная UTM-строка с подсветкой синтаксиса
- Кнопка «Скопировать»
- Кнопка «Сгенерировать ещё»
